# Dearly AI

A library to easily use generative AI tools.

## Setup

1. `git clone https://github.com/leekycauldron/dearly-ai.git`
2. `cd dearly-ai`
3. `pip install -e .`
4. Set `OPENAI_API_KEY` in `.env.example` and rename file to `.env`
